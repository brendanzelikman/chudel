Welcome to the Chudel library for ChucK!

File Breakdown:

    demo.ck: An example of how to use Chudel
    - Loads a sequence and runs the engine

    chudel.ck: The main Chudel class / API
    - Handles scheduling and playback

    parser.ck: The Mini Notation parser
    - Responsible for converting strings into patterns

    pattern.ck: The core logic for sequencing events
    - Contains a class for each kind of operation

    hap.ck: The events that capture musical actions
    - Each class has a map with all kinds of keys and values

    recorder.ck: A class to record audio output
    - Sourced from the ChucK examples

    sampler.ck: A class to hold sound buffers
    - Loads and plays samples with an effect chain

    scale.ck: A class for parsing scales and notes
    - Adapted from another project

    utils.ck: A class for various utility functions
    - Provides help with strings, numbers, etc.

    examples.ck: Preset strings for musical examples
    - Used to save space in other files

    samples/: A folder containing audio samples
    - Note that custom samples can be used with chudel.register()

    chudel.txt: A file used for livecoding
    - This is the default file that will be called with chudel.livecode()

    explanation.txt: This file!
